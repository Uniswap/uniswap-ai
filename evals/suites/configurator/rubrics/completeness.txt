# Completeness Rubric for CCA Configurator

Evaluate whether the generated output includes all expected elements and follows best practices.

## Expected Elements (Should Include)

Score based on how many of these are present:

1. **Configuration Flow**
   - Produces a complete configuration with all required parameters
   - Validates inputs and shows any warnings
   - Shows a summary of the configuration

2. **Network-Specific Constants**
   - Correct block time for selected network
   - Correct USDC address for selected network
   - Correct RPC endpoint

3. **Price Calculation Explanation**
   - Explains Q96 fixed-point format
   - Shows decimal adjustment calculation
   - Demonstrates floor price rounding

4. **Supply Schedule Details**
   - Shows MPS values for each step
   - Shows block durations for each step
   - Includes summary statistics (total MPS, final block percentage)

5. **Validation Checklist**
   - Block sequence validation
   - Floor price divisibility check
   - Supply schedule total verification
   - Address format validation

6. **Educational Context**
   - Explains CCA auction mechanics
   - Provides parameter guidance and recommendations
   - References documentation resources

## Penalties (Should Not Include)

Deduct points for:
- Exposing or requesting private keys directly (-0.15)
- Incorrect decimal adjustment for currency (-0.15)
- Supply schedule not summing to 10,000,000 MPS (-0.15)
- Missing floor price divisibility verification (-0.1)

## Scoring Guide

Base score from expected elements:
- 6/6 elements: 1.0
- 5/6 elements: 0.9
- 4/6 elements: 0.8
- 3/6 elements: 0.65
- 2/6 elements: 0.5
- 1/6 elements: 0.3
- 0/6 elements: 0.0

Then subtract any penalties.

## Instructions

Count the expected elements present, then apply any penalties for issues found. Provide the final score.
