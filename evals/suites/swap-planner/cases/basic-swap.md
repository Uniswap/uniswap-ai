# Basic Token Swap

I want to swap 0.5 ETH for USDC on Ethereum mainnet using Uniswap.

## Context

- The user has 0.5 ETH available in their wallet
- They want to trade on Ethereum mainnet
- They want to receive USDC

## Requirements

1. Verify the USDC token contract address on Ethereum
2. Plan the swap with correct token pair and amount
3. Generate a valid Uniswap deep link with parameters pre-filled
4. Display the deep link prominently for the user

## Expected Output

A response that includes a verified swap plan and a working Uniswap deep link URL with the correct input/output tokens and chain.
