## Token Address Lookup

When a user provides a token symbol, first check the common tokens list above. If not found:

1. Use web search: `"{symbol} token contract address {chain}"`
2. Verify the address on the chain's block explorer
3. Check if the token has liquidity on Uniswap

## Native Token Handling

For native tokens (ETH, MATIC, BNB, AVAX, CELO):

- Use `NATIVE` as the `inputCurrency` or `outputCurrency` parameter
- Do NOT use the wrapped token address for native tokens in swap URLs

**Examples:**

- Swap ETH → USDC: `inputCurrency=NATIVE`
- Swap WETH → USDC: `inputCurrency=0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2`

## RPC Rate Limits

Public RPCs have rate limits. For basic contract verification, public RPCs are sufficient. For production applications, consider using:

- Alchemy
- Infura
- QuickNode
- Or chain-specific premium RPCs
